# Foreign Fortran

Run the exact same Fortran code with eight different approaches
from five different programming languages:

- [Fortran][1]
- [C][2]
- [C++][8]
- Python with [`ctypes`][3]
- Python with [`cffi`][4]
- Python with [`f2py`][5]
- [Cython][6]
- [Go (Golang)][7]
