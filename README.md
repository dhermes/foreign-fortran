# Foreign Fortran

Run the exact same Fortran code with six different approaches
from three different programming languages:

- [Fortran][1]
- [C][2]
- Python with [`ctypes`][3]
- Python with [`cffi`][4]
- Python with [`f2py`][5]
- [Cython][6]
